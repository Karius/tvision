## This directory contains tools used only on testing the code, some log files and tests results/statistics used od development stage.

- ```optimisation.log```

  Statistics about text, bss and data section size tvision library and all demo programs generated using different optiimisation methods.

- ```optimisation.sh```

  Script which generates optimisation.log.

- ```warnings-doxygen.log```

  Current list of warnings generated by doxygen during generate tvision html documentation.

- ```warnings-clang.log```

  Current stattistis about compile warnings and raw log of all compaile warnings generated using clang.
- ```warnings-gcc.log```

  Current stattistis about compile warnings and raw log of all compaile warnings generated using gcc.

- ```warnings.sh```

  Script which generates ```warnimgs*.log```.
